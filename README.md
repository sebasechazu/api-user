# API de Usuarios - CRUD

## Descripción

Esta API permite gestionar una lista de usuarios a través de operaciones CRUD (Crear, Leer, Actualizar y Eliminar). Está construida utilizando Node.js y varias librerías para garantizar la seguridad y eficiencia en el manejo de los datos.

## Librerías Utilizadas

- **bcryptjs**: ^2.4.3 - Para la encriptación de contraseñas.
- **cors**: ^2.8.5 - Para permitir solicitudes desde diferentes orígenes.
- **express**: ^4.19.2 - Framework para la creación de aplicaciones web y APIs.
- **jwt-simple**: ^0.5.6 - Para la creación y manejo de JSON Web Tokens.
- **moment**: ^2.30.1 - Para la manipulación y formateo de fechas.
- **mongodb**: ^6.7.0 - Driver para la base de datos MongoDB.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/api-usuarios-crud.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd api-usuarios-crud
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```

## Uso

1. Inicia el servidor:
   ```bash
   npm start
   ```
2. Accede a la API en `http://localhost:3000`.

## Endpoints

- `GET /users`: Obtener la lista de usuarios.
- `POST /users`: Crear un nuevo usuario.
- `GET /users/:id`: Obtener un usuario por ID.
- `PUT /users/:id`: Actualizar un usuario por ID.
- `DELETE /users/:id`: Eliminar un usuario por ID.
